# Collaborative Whiteboard

A real-time collaborative whiteboard application built with the MERN stack and Socket.io.

## Setup

1. **Install dependencies:**
   ```bash
   cd server && npm install
   cd client && npm install
   ```

2. **Start MongoDB:**
   Ensure MongoDB is running locally on default port 27017

3. **Run the application:**
   ```bash
   cd server && npm start
   cd client && npm start
   ```

## Features
- Join/create rooms with simple room codes
- Real-time drawing with pencil tool
- Color selection and stroke width adjustment
- Real-time cursor tracking
- User presence indicator
- Clear canvas functionality
- Save as PDF functionality

## Technologies
- MongoDB
- Express.js
- React.js
- Node.js
- Socket.io
